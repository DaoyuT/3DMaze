
In Linux, try command 
"gcc -o maze Maze.cpp -lGL -lGLU -lglut -lm

" and 
"gcc -o maze MazeEZ.cpp -lGL -lGLU -lglut -lm

" to run the project.

Maze.cpp is the original version, it might lag a bit, but you can also try MazeEZ.cpp which is a easy version, it runs faster, buy some wall may disappear sometime.

Operations:
     forward: 	w   
		backward:	s  
		leftward:	a  
		rightward:	d   
		turn left:	l  
		turn right:	j
   
		float:		z   
		sink:		x

Hint: mouse click might help :)
